Unipag.py - Unipag Client for Python

.. image:: https://travis-ci.org/unipag/unipag-python.png?branch=master

Installation
------------

Install using pip (recommended): ::

    $ pip install unipag

or using easy_install: ::

    $ easy_install unipag
